Stat Abbreviation Definition/Explanation
G.P This is an abbreviation for Games Played, meaning the number of games the corresponding team.
A.S.O.N This abbreviation is for average shots on net, meaning the the number of times the goalie has to stop the puck from going in the net, or if the shot goes into the net.
A.H This abbreviation is for the average number of hits a game for the specified team.
P.P This is an abbreviation for Puck Percentage (averaged), this is an indicator for how much on average a team has the puck.
Save % This is simply the save percentage.
